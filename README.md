# Button Catalog

iOS 15 から利用できる UIButton.Configuration やトグルボタン、ポップアップボタンなど使ってみるための実験プロジェクトです。

## コンテンツ

- [x] 4つのデフォルトスタイル
    - plain
    - gray
    - Tinted
    - Filled
- [x] ボタンサイズ
- [x] 画像位置
- [x] Subtitle
- [x] 複数行
- [x] 背景カスタム
- 手動で update
- ActivityIndicator
- Toggle Button
- Pull-down
- Pop-up
